# 🛒 마트어택맵 (Mart Attack Map)

### 👉 [앱 바로 실행하기](https://lgw7126.github.io/Claude-Mart-Attack-Map-app/) 👈

> 여행지 마트에서 구경만 하다 나오지 마세요.
> 도시를 입력하면 **현지인 추천 마트 필수템 Top 10**을 보여주고, 산 것을 하나씩 체크할 수 있는 웹앱입니다.

<sub>⚙️ 위 링크가 404라면 GitHub Pages를 한 번만 켜주세요: 저장소 **Settings → Pages → Branch를 `main`으로 선택 → Save** (1~2분 뒤 접속 가능)</sub>

## 🎯 해결하는 불편

- **문제**: 여행지 현지 마트·편의점에서 뭘 사야 할지 몰라 구경만 하다 나온다.
- **타깃**: 현지 식문화를 진짜로 경험하고 싶은 20~30대 여행자.
- **MVP 기능**: 도시 검색 → 필수템 Top 10 리스트 → 구매 체크 (새로고침해도 유지!)

## 🚀 실행 방법 (설치 필요 없음!)

이 앱은 서버나 프로그램 설치 없이 브라우저만 있으면 됩니다.

1. 이 저장소를 다운로드(또는 `git clone`)합니다.
2. `index.html` 파일을 **더블클릭**해서 브라우저로 엽니다.
3. 끝! 도시를 검색하고 필수템을 체크하세요.

로컬 서버로 열고 싶다면 (선택사항):

```bash
# 프로젝트 폴더에서 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 🗂️ 파일 구조

| 파일 | 역할 |
|------|------|
| `index.html` | 화면 뼈대 (검색창, 결과 영역) |
| `style.css` | 디자인 (색상, 카드, 진행률 바) |
| `data.js` | 도시별 필수템 Top 10 데이터 (앱의 "데이터베이스") |
| `app.js` | 동작 로직 (검색, 체크 저장, 화면 그리기) |

## 👤 내 정보 기능

- 닉네임을 저장하면 상단에 **"OO님의 마트어택"** 카드가 표시됩니다.
- 모든 도시에서 체크한 아이템 수와 **정복한 도시 수**가 자동 집계됩니다.
- 닉네임과 체크 기록은 모두 이 브라우저(localStorage)에만 저장됩니다.

## 🏙️ 도시 추가하는 법

`data.js`를 열고 `CITY_DATA` 배열에 아래 형식으로 객체를 추가하면 끝입니다.

```js
{
  id: "fukuoka",                    // 고유한 영문 키 (저장에 사용)
  name: "후쿠오카",                  // 화면에 표시할 이름
  country: "일본 🇯🇵",
  aliases: ["후쿠오카", "fukuoka"],  // 검색어로 인정할 단어들
  marts: "돈키호테 · 마루쿄",
  items: [
    { emoji: "🍜", name: "상품명", desc: "설명", where: "파는 곳", price: "가격" },
    // ... 10개 작성
  ],
},
```

## 🔧 어떻게 동작하나요? (초보자용)

1. 도시를 입력하면 `app.js`의 `findCity()`가 `data.js`에서 도시를 찾습니다.
2. `renderCity()`가 Top 10 리스트를 HTML로 만들어 화면에 그립니다.
3. 항목을 체크하면 브라우저의 **localStorage**에 저장됩니다.
   → 앱을 껐다 켜도, 새로고침해도 체크 상태가 그대로 남습니다.
4. 10개를 모두 체크하면 🎉 어택 완료 배너가 나타납니다.
